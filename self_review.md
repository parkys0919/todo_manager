# Self Review

## What I Did Well

In this project, I successfully applied object-oriented programming concepts such as classes, inheritance, and method overriding. I organized the project as a Python package and separated responsibilities into different modules, which improved readability and maintainability. I also implemented automated testing with pytest and ensured that all tests passed successfully.

## Areas for Improvement

One limitation of this project is that the functionality is relatively simple compared to real-world task management applications. The package currently stores data only in CSV format and does not provide advanced features such as graphical interfaces, reminders, or database integration.

## Future Improvements

If I continue developing this project, I would like to add more advanced task management features such as categories, notifications, and task filtering. I would also like to improve the recurring task system and integrate a database backend to manage large amounts of task data more efficiently.
